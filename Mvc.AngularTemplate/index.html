.Net MVC4 Angular Template

This template will help you jumpstart your client-side angular mvc code using server-side .Net WebAPI.
The following steps are needed to get started (Explanation of tools used). If this is not your first time using these tools, skip to step 4.
1. Install NodeJs
[Node.js](http://nodejs.org) is a platform based on Chrome's JavaScript runtime, called [V8](http://code.google.com/p/v8/). It allows you to develop all kinds of software using JavaScript. A great feature of Node.js is its wide variety of existing libraries and tools. As the developer community is absolutely massive and incredibly active, Node.js has a basic package manager called NPM that you can use to install Node.js-based software and libraries from the command line. While `MVC-NG-Boilderplate` makes heavy use of Node.js behind the scenes, you as the application developer don't need to really think about it much. Most of the interaction with Node.js will occur through Grunt, so you really only need to know how get the initial setup working.
2. Install GIT for Windows
[Git](http://git-scm.com/) is a distributed version control system. `MVC-NG-Boilerplate` uses git to manage its codebase. While in theory you don't have to use Git once you download `MVC-NG-Boilerplate`, this project makes the assumption that you do. If you're on GitHub, I assume you already have a basic understanding of Git, which is all you need to make effective use of this project. You just need to be able to commit and push and junk - nothing funky. If you're not familiar with it, check out the documentation linked to above. GitHub also has a great [help section](https://help.github.com/). Bower package manager makes use of this to install packages.
3. Change GIT download options
Run the following command to change the download options for git to https which will help with firewall protocol issues. 
git config --global url."https://".insteadOf git://
4. Install required tools global
Run the following command in CMD or Git-Bash 
npm install -g grunt-cli karma bower
5. Install package.json dependencies
Run the following command in CMD or Git-Bash in the project folder 
npm install
6. Install bower.json dependencies
Run the following command in CMD or Git-Bash in the project folder 
bower install
7. Run Grunt to build and compile your application
Run the following command in CMD or Git-Bash in the project folder: For continuous building 
grunt watch
To just build 
grunt build
To compile for production release (post building) 
grunt compile
To build and compile 
grunt
